# Async Data Platform

Production-grade backend built using:

- Java 21
- Spring Boot 3
- PostgreSQL
- Redis
- Flyway
- Docker
- Spring Data JPA
- Spring Security (WIP)
- JWT (WIP)

## Features

### Completed

- User Management API
- PostgreSQL Integration
- Flyway Database Migrations
- Dockerized Infrastructure
- Redis Integration
- Spring Actuator

### Planned

- JWT Authentication
- Async Job Processing
- File Uploads
- Retry Framework
- Metrics & Monitoring
- AWS Deployment

## Run

docker compose up -d

./mvnw spring-boot:run
